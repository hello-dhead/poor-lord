日本語

モンスター８体のスプライトパッケージです。
DEMO内で動作する、スクリプトを内包します。
これは、rigidbody2Dで当たり判定を行い、シンプルなSpriteアニメーションをスクリプトで実装します。
そして、シンプルなアルゴリズムでモンスターたちは動いています。

あなたのプロジェクトで使用するためには、少しの改変が必要になるでしょう。
そのまま動作するとは考えないでください。

サンプルシーン内、プレイヤーはFairyを操作できます。WASDで移動ができ、Spaceで攻撃ができます。
モンスターに攻撃がヒットすれば、ダメージグラフィックを確認できます。
Fairyがモンスター、もしくはモンスターの攻撃にに触れれば、青く光ります。（これはダメージを受けたということです。）

あなたのプロジェクトのキャラクターが、このルールをどう処理するかはあなた次第です。
うまく作り変えてください。

--------
English

This is a sprite package for eight monsters.
Contains a script that runs in the DEMO.
It uses rigidbody2D to determine the hits and implements a simple Sprite animation in a script.
And the monsters are moving with a simple algorithm.

It will need a few modifications to be used in your project.
Don't expect it to work as is.

In the sample scene, the player can control the Fairy, using WASD to move and Space to attack.
If a monster is hit, you can see the damage graphic.
If the Fairy touches the monster or the monster's attack, it will glow blue. (This means you've taken damage.)

It's up to you how the characters in your project handle this rule.
Make it up as you go along.
